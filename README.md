# auto-derby

自动化养马

功能

- [x] 团队赛
  - [x] 有胜利确定奖励时吃帕菲
- [x] 日常赛 (当前第一名以外无法识别 因为没图)
- [x] 传奇赛
  - [x] 自动领奖励
- [x] 育成
  - [x] 根据当前属性选择训练
  - [ ] 根据训练效果选择训练
  - [ ] 给暑假集训保留体力
  - [ ] 自动出赛适应性匹配的 G1

使用方法

需求 DMM 版 和 python3.8

```shell
py -3.8 -m pip install -r requirements.txt
py -3.8 -m auto_derby
```
